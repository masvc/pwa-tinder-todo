# Swipe Todo

Tinder風スワイプUIでタスクを管理するPWAアプリ。

## 使い方

カードをスワイプしてタスクのステータスを変更:

| 方向 | アクション |
|------|-----------|
| ← | スキップ（後ろに回す） |
| → | 進行中 |
| ↑ | 完了 |
| ↓ | アーカイブ |

## 機能

- 優先度（高/中/低）で色分け
- カードスタックで残りタスクを可視化
- オフライン対応（PWA）
- ローカルストレージで永続化

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```
