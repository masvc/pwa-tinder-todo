# Swipe Todo

Tinder風スワイプUIのタスク管理アプリ

> **注意: これはプロトタイプです**
>
> APIキーはブラウザのlocalStorageに保存されます。本番環境での使用は推奨しません。

## スワイプ操作

| 方向 | アクション |
|------|-----------|
| ← | スキップ |
| → | 進行中 |
| ↑ | 完了 |
| ↓ | アーカイブ |

## 機能

- 優先度設定（高/中/低）
- AIタスク生成（Claude API）
- データのエクスポート/インポート
- PWA対応

## 開発

```bash
npm install
npm run dev
```
